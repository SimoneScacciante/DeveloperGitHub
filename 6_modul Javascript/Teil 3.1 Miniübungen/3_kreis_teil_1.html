Aufgabe 3

3. Wir wollen HTML Code generieren! Der nachfolgende Button generiert einen Kreis. Probiere ihn aus!

3.1. Unser Kreis soll schöner aussehen: Füge ihm die CSS Eigenschaft box-shadow: -4px 9px 10px rgba(0, 0, 0,0.4); hinzu. Tipp: Hierzu brauchst du kein JavaScript.
3.2. Unser Kreis soll nicht einfach nur "Kreis" heißen. Benenne das Wort "Kreis" um in einen Namen, z.B. Claus.
3.3. Damit der Name deutlicher zu sehen ist, sollte er Fett geschrieben werden. Verwende dafür das <b>-Tag und füge den Namen in die Mitte ein.
